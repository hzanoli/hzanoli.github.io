Running containerized X11 apps on MacOS

Mon 19 April 2021
By Henrique J C Zanoli
Category: Programming
Tags: Docker Containers MacOS

When developing apps in (docker) containers, you might like to run an X11 application. This is mostly straightforward on Linux, but requires a bit of work on MacOS

First, install XQuartz. There is a .dmg package in the Download section of the website. If you are like me and would rather prefer to install from the command line, you surely like Homebrew and you can use it to install XQuartz:

brew install xquartz
